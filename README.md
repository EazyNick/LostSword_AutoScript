## 사용 방법

### 워크플로우 생성
1. 웹 UI에서 "노드 추가" 버튼을 클릭하거나 `Ctrl+N` 단축키를 활용해 노드를 생성합니다.
2. 원하는 노드 타입을 선택합니다 (클릭, 이미지 터치, 대기 등)
3. 노드를 캔버스에 배치하고 연결합니다
4. 각 노드의 설정을 조정합니다 (노드 더블클릭 또는 노드 우측 상단의 설정 버튼 클릭)

### 워크플로우 저장
- `Ctrl+S` 키를 누르거나 저장 버튼을 클릭합니다
- 워크플로우 이름을 입력하고 저장합니다

### 워크플로우 실행
- `F5` 또는 `Ctrl+R` 키를 누르거나 실행 버튼을 클릭합니다
- 실행 중인 노드는 주황색으로 표시되고, 완료된 노드는 초록색으로 표시됩니다
- 에러가 발생한 노드는 빨간색으로 표시되고 실행이 중단됩니다
- 실행 결과: 성공/실패/중단 노드 개수가 표시됩니다
- **전체 실행**: 헤더의 "전체 실행" 버튼을 클릭하면 활성화된 모든 스크립트가 순차적으로 실행됩니다
- **스크립트 활성화/비활성화**: 대시보드에서 각 스크립트 카드의 활성/비활성 버튼을 클릭하여 실행 여부를 제어할 수 있습니다

### 키보드 단축키
- `Ctrl+N`: 노드 추가
- `Ctrl+S`: 워크플로우 저장
- `Ctrl+Z`: 실행 취소
- `Ctrl+Y`: 다시 실행
- `F5` / `Ctrl+R`: 워크플로우 실행
- `Delete`: 선택한 노드 삭제
- `Esc`: 실행 중지

## 주요 기능

- **시각적 워크플로우 편집기**: 드래그 앤 드롭으로 노드 기반 스크립트 생성
- **다양한 노드 타입**: 클릭, 이미지 터치, 대기, 조건 분기, 루프 등
- **실시간 실행 모니터링**: 실행 중인 노드를 시각적으로 확인
- **워크플로우 저장/로드**: 생성한 워크플로우를 저장하고 나중에 불러오기
- **대시보드**: 스크립트 통계 및 관리 (전체 스크립트, 오늘 실행 횟수, 오늘 실패한 스크립트, 비활성 스크립트)
- **스크립트 활성화/비활성화**: 대시보드에서 스크립트별로 활성화 상태를 관리하여 "전체 실행" 시 실행 여부를 제어
- **설정 페이지**: 테마, 실행 설정, 스크린샷, 알림 등 다양한 설정 관리
- **라이트/다크 모드**: 시스템 설정에 따른 자동 테마 전환 지원
- **SPA 구조**: 페이지 전환 없이 부드러운 사용자 경험
- **실행 기록 관리**: 스크립트 실행 이력을 데이터베이스에 저장하여 통계 및 분석 지원
