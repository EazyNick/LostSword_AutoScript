# 노드 목록

AutoScript에서 사용 가능한 모든 노드 타입을 설명합니다.

## 경계 노드 (Boundary Nodes)

### 시작 노드 (Start)
- **설명**: 워크플로우의 시작점을 나타내는 노드
- **용도**: 워크플로우 실행 시 가장 먼저 실행되는 노드

### 종료 노드 (End)
- **설명**: 워크플로우의 종료점을 나타내는 노드
- **용도**: 워크플로우 실행을 종료하는 노드

## 액션 노드 (Action Nodes)

### 액션 노드 (Action)
- **설명**: 기본 액션을 수행하는 노드
- **용도**: 범용 액션 노드로 사용자 정의 액션 구현 가능

### 클릭 노드 (Click)
- **설명**: 화면의 특정 좌표를 클릭하는 노드
- **용도**: 지정된 X, Y 좌표를 클릭

### 전투 노드 (Battle)
- **설명**: 전투를 수행하는 노드
- **용도**: 게임 내 전투 액션 실행

### 수집 노드 (Collect)
- **설명**: 아이템을 수집하는 노드
- **용도**: 게임 내 아이템 수집 액션 실행

### 이동 노드 (Move)
- **설명**: 캐릭터를 이동시키는 노드
- **용도**: 캐릭터를 지정된 위치로 이동

### 네비게이션 노드 (Navigate)
- **설명**: 목적지로 이동하는 노드
- **용도**: 특정 목적지로 자동 이동

### HTTP API 요청 노드 (HTTP API Request)
- **설명**: 외부 API에 HTTP 요청을 보내는 노드
- **용도**: RESTful API 호출 및 데이터 통신

### 프로세스 포커스 노드 (Process Focus)
- **설명**: 선택한 프로세스의 창을 화면 최상단에 포커스하는 노드
- **용도**: 특정 애플리케이션 창을 활성화

### 대기 노드 (Wait)
- **설명**: 지정된 시간만큼 대기하는 노드
- **용도**: 워크플로우 실행 중 일정 시간 대기

## 이미지 노드 (Image Nodes)

### 이미지 터치 노드 (Image Touch)
- **설명**: 화면에서 이미지를 찾아 터치하는 노드
- **용도**: 화면 캡처를 통해 특정 이미지를 찾아 클릭
- **특징**: 이미지 파일 경로 설정 필요

## 로직 노드 (Logic Nodes)

### 조건 노드 (Condition)
- **설명**: 조건을 확인하고 결과를 반환하는 노드
- **용도**: 조건에 따라 워크플로우 분기 처리

### 반복 노드 (Loop)
- **설명**: 반복 작업을 수행하는 노드
- **용도**: 특정 작업을 반복 실행

## 노드 카테고리

- **시스템 노드**: 시작, 종료 노드
- **액션 노드**: 클릭, 이동, 전투 등 실행 액션
- **로직 노드**: 조건 분기, 반복 처리
- **이미지 노드**: 이미지 인식 및 터치

## 참고

- 새로운 노드를 추가하려면 [노드 추가 가이드](dev/creating-nodes.md)를 참고하세요.
- 각 노드의 상세 설정은 워크플로우 편집기에서 노드를 더블클릭하여 확인할 수 있습니다.
