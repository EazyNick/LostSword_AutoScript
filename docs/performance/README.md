- **내용**: 프론트엔드 성능 최적화 가이드
- **주요 내용**:

### 데이터베이스 최적화
- **파일**: `database-optimization.md`

  - 성능 모니터링 (쿼리 실행 시간 측정, 느린 쿼리 로깅)


## 성능 최적화 가이드라인

### 일반적인 최적화 원칙

1. **리플로우 최소화**
   - 레이아웃 정보를 읽는 API 호출 최소화
   - 결과를 캐싱하여 재사용
   - 스타일 변경을 배치 처리

2. **렌더링 최적화**
   - `requestAnimationFrame` 사용
   - 불필요한 업데이트 제거
   - 스로틀링/디바운싱 적용

3. **메모리 관리**
   - DOM 파싱 대신 메모리 객체 사용
   - 캐시 무효화 전략 수립
   - 이벤트 리스너 정리
