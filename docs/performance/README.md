# 성능 최적화 문서

이 폴더는 프로젝트의 성능 최적화 및 개선 과정에 대한 문서를 담고 있습니다.

## 문서 목록

### 좌표 계산 최적화
- **파일**: `coordinate-calculation-optimization.md`
- **내용**: 캔버스 내 위치 이동 및 노드 드래그 시 발생하던 렉을 해결하기 위한 좌표 계산 로직 최적화
- **주요 개선 사항**:
  - `getBoundingClientRect()` 캐싱으로 리플로우 최소화
  - `requestAnimationFrame`을 사용한 업데이트 스로틀링
  - 연결선 업데이트 스로틀링
  - 메모리 객체 활용으로 DOM 파싱 제거

## 성능 최적화 가이드라인

### 일반적인 최적화 원칙

1. **리플로우 최소화**
   - 레이아웃 정보를 읽는 API 호출 최소화
   - 결과를 캐싱하여 재사용
   - 스타일 변경을 배치 처리

2. **렌더링 최적화**
   - `requestAnimationFrame` 사용
   - 불필요한 업데이트 제거
   - 스로틀링/디바운싱 적용

3. **메모리 관리**
   - DOM 파싱 대신 메모리 객체 사용
   - 캐시 무효화 전략 수립
   - 이벤트 리스너 정리
